HTML

language written in form of elements consisting in tags enclosed in
<angle brackets>

HTML tags

  • Links: <a>
  • Images: <img>
  • Lists: <ul>, <ol> and <li>
  • Headings: <h1>, <h2> ... <h6>
  • Paragraph: <p>
  • Emphasized text: <strong>, <em>
  • Divisions: <div>
  • Tabular data: <table>, <tr>, <td>

Common element attributes

aka global attributes

  • class: space-separated list of the classes of the element
  • data-*: data attributes, allow proprietary information to be exchanged between the HTML and its DOM
  • id: unique identifier which must be unique in the whole document. Its purpose is to identify the element when linking.
  • style: contains CSS styling declarations to be applied to the element.
  • title: contains a text representing advisory information related to the element it belongs to.
  • events: onload, onmousedown, onmouseover etc
  • ...
