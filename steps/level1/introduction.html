Hypertext

Text that contains links to other texts.

Coined by Ted Nelson around 1965

Markup language

  • a language that uses instructions to semantically describe its nature.
  • a system for annotating a document in a way that is syntactically distinguishable from the text.
  • not a programming language.
  • evolved from the "marking up" of paper manuscripts.
  • like TeX, XML or SVG.
