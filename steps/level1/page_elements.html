Link element attributes

  • target: specifies where to display the linked resource: _same, _blank, _parent, _top

Images

the img tag



    <img src="http://www.google.it/logo.png" alt="Google logo" width="120" height="80" />

represents an image in the document. Example of self-closing tag, like <br>

Image element attributes

  • src: mandatory image url.
  • alt: defines the alternative text describing the image
  • width, height: define the size of the image in pizels, optional.

Text formatting elements

  • <b> bold text
  • <em> emphasized text
  • <i> italic text
  • <small> smaller text
  • <strong> important text
  • <sub> subscripted text
  • <sup> superscripted text
  • <ins> inserted text
  • <del> deleted text
  • <mark> marked/highlighted text
