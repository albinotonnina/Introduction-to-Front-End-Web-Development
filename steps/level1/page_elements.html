Images

the img tag



    <img src="http://www.google.it/logo.png" alt="Google logo" width="120" height="80" />

represents an image in the document. Example of self-closing tag, like <br>

Image element attributes

  • src: mandatory image url.
  • alt: defines the alternative text describing the image
  • width, height: define the size of the image in pizels, optional.

Div

The document division element


    

        

            ...
        

        ...

  • generic container for flow content
  • does not inherently represent anything
  • can be used to group elements for styling purposes (using the class or id attributes)

divs as structural elements

  • Header at the top of the page, usually containing the top level heading of the page, and/or logos.
  • Main content column, which holds the main content of functionality the user is there to use.
  • One of more sidebars, holding tributary content that is either related to that page's main content or relevant anyway.
  • Navigation menu going across the page or down the page in a column. This often put in a sidebar, or may form part of the header.
  • A footer that goes across the bottom of the site and contains secondary information.
