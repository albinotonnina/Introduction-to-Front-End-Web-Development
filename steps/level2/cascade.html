cascading style sheets

The cascade is a fundamental feature of CSS.

It is an algorithm defining how to combine properties values originating from different sources.

Cascade: Author styles

ordered by importance

  •      <link rel="stylesheet" type="text/css" href="styles.css" />
    External styles are styles that are written in a separate document. 
  •      <style type="text/css"> p { color: #00f; } </style>
    Embedded styles are styles that are embedded in the head of the document. 
  •      <a href="#" style="text-decoration: none; color: blue;">
    Inline styles are styles that are written directly in the tag on the document.
