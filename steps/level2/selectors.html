CSS Selectors

CSS selectors are used to find (or select ) HTML elements based on their name, id, class, attribute, and more.

Id based selectors

        CONTENT

Grouping


    h1 {
        text-align: center;
        color: red;
    }

    h2 {
        text-align: center;
        color: red;
    }

    p {
        text-align: center;
        color: red;
    }
                

    h1, h2, p {
        text-align: center;
        color: red;
    }

        padding:0;
